A projektről és a technológiáról

A projektről

Ez az alkalmazás egy tanulási projekt, amely bemutatja hogyan lehet egy Flask webappot Dockerrel, Nginx-szel és GitHub Actions CI/CD pipeline-nal üzemeltetni AWS EC2-n.

🐍 Python + Flask

A backend Python Flask keretrendszerrel készült. Könnyű, gyors és egyszerűen tanulható.

🖥️ AWS EC2

Az alkalmazás Amazon Web Services EC2 t3.micro instance-on fut, amely az ingyenes szinten elérhető.

🔀 Nginx

Az Nginx reverse proxyként működik, és a kéréseket a Flask alkalmazáshoz irányítja.
